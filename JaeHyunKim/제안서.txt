## 제안서 
1. 제안목적
	- IT 관련 교육생 및 개발자 등 토이프로젝트에 대한 수요는 많음 
	- 현재 뚜렷한 토이프로젝트 관련 플랫폼 미존재
		>> 네이버 카페, 취업사이트, IT 커뮤니티에서 별도 게시판을 통해 토이프로젝트 진행
		>> 채널이 너무 다양하고 산발적이어서 프로젝트를 찾거나 알리는데 번거로움 
		
2. 주요 고객 
	- IT 관련 교육중인 수강생
	- IT 관련 취업준비생
	- 회사 이직을 준비중인 분
	- IT 인재 채용을 희망하는 분
	- 프로젝트 팀원을 모집하려는 분
   
3. 어떻게 서비스를 제공할 것인가
	- Web 기반 서비스 제공
	- 두 그룹군에 대한 플랫폼 제공
		- 프로젝트 모집군 : 프로젝트 관련 설명 및 필요인력 소개
		- 프로젝트 지원군 : 본인 전공 선택 및 정보공개 동의시 모집군에 노출 
	- 게시판과 DM 등을 통한 커뮤니케이션 장소 제공
	- 투표 기능을 통한 프로젝트 및 발주자, 개발자에 대한 평가
	- 신고 기능을 통한 누적신고 프로젝트 및 발주자, 개발자에 대한 블라인드 처리

4. 기능 우선 순위
	- 프로젝트 모집 게시판
		>> 무료, 유료 프로젝트 모집
		>> 무료 : 모집 게시판 검색, 읽기 가능
		>> 유료 : 모집 게시판 검색, 읽기, 참여하기 가능
	- 프로젝트 팀원 찾기
		>> 회원가입시 본인의 전공분야, 지원가능 상태 체크시 화면에 노출
	- 투표 기능
		>> 투표 기능을 통해 회원들이 관심있어하는 프로젝트 순위별로 메인 화면에 노출
	- 신고 기능
		>> 신고 들어온 게시물에 대한 블라인드 처리 및 관리자 검수 기능
	- 회원정보관리
		>> 로그인
		>> 회원가입
		>> 아이디찾기
		>> 비밀번호찾기/변경
		>> 마이페이지
			>> 회원정보 변경
			>> 회원탈퇴
