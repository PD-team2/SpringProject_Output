- 비로그인 : FAQ 게시판 가능
- 로그인 : FAQ, 1:1 게시판 가능

## 마이페이지
- 비로그인 : 진입불가능
- 로그인 : 메인메뉴에서 마이페이지 진입가능
	선행 조건 : 로그인한 회원이 마이페이지 진입
	작동 : 해당 페이지 기능들 수행

-- 삭제될 기능 -- 	
1. 프로필 사진 변경
	기본값 : 사진없음 
	작동 : 프로필 수정 버튼 누른 후 사진 파일 업로드시 변경
	
삭제 사유 : 로그인시 입력받는 프로필 사진과 중첩

2. 로그인기록
	작동: Spring Security ip 저장기능을 통해 접속시간, 접속아이피 값 출력
	
삭제 사유 : 윤아님 로그인 작업과 코드 충돌 가능성 높음 다른 기능에 집중
