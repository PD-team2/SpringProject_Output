- 비로그인 : FAQ 게시판 가능
- 로그인 : FAQ, 1:1 게시판 가능

## 마이페이지
- 비로그인 : 진입불가능
- 로그인 : 메인메뉴에서 마이페이지 진입가능
	선행 조건 : 로그인한 회원이 마이페이지 진입
	작동 : 해당 페이지 기능들 수행
	
1. 프로필 사진 변경
	기본값: 사진없음 
	작동: 프로필 수정 버튼 누른 후 사진 파일 업로드시 변경

2. 로그인기록
	작동: Spring Security ip 저장기능을 통해 접속시간, 접속아이피 값 출력
	
3. 내활동 기록
	>> [주리님 작업페이지로 이동]
		
4. 계정관리
	>> [낙원님 작업페이지로 이동]
	>> 비밀번호 변경 [주리님 페이지로 이동]
	
5. 1:1 문의
	작동 : 1:1 문의 페이지로 이동(1,2,3 중에 실현 가능한 방식으로 구현)
	1. 운영진에게 이메일 발송(EmailJS API 사용)
	2. 댓글/대댓글 방식 문의 (DISQUS API 사용)
	3. 웹 소켓(Web Socket) 이용한 채팅방식
	
6. 회원탈퇴 [윤아님 작업페이지로 이동]
