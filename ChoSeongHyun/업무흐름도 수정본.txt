<성현 기능 목록 : 회원가입, 팀원찾기 게시판 + a>

(1) 회원 가입 페이지	 
- 비로그인 : 회원 가입 가능
- 로그인 : 회원 가입 불가능 (이용불가 메시지 팝업)

	(1-1) 필수 입력 데이터 설정
	ㄴ 	아이디, 비밀번호, 이름, 핸드폰번호, 이메일, 관심분야(프론트엔드/백엔드/기획/마케팅/디자인/기타)
	ㄴ	선행 조건 : 회원 유무 체크
	ㄴ	작동 : 입력데이터 공백일 경우 폼 제출 불가능

	(1-2) 중복 체크 및 기타 조건 
	ㄴ	아이디 중복 체크
	ㄴ	아이디 : 길이 제한 6~20자리/특수문자 불가능
		비밀번호 : 길이 제한 8~20자리/특수문자 !@#$%^&*_- 사용가능
		핸드폰 번호 : 양식 000-0000-0000 (인증을 할수있게...??...)
		이메일 : 아이디 @네이버/지메일/한메일/네이트/직접입력 (인증을 할수있게...??...)
		관심분야 : 프론트엔드/백엔드/기획/마케팅/디자인/기타 체크박스 다중 허용

	(1-3) 폼 제출 후 로그인 페이지로 이동
	ㄴ	선행 조건 : 회원가입 폼을 작성 완료한 상태
	ㄴ	작동 : <완료> 버튼 클릭 시 데이터가 정상적으로 입력되었는 지 확인 후 로그인 페이지로 연결

	(1-4) 회원 DB 암호화
	ㄴ	선행 조건 : 사용자가 회원 가입 폼을 제출한 상태
	ㄴ	작동 : SHA256으로 비밀번호 암호화 ? ...


(2) 프로젝트 팀원 찾기 목록 페이지 
- 회원의 자기 소개글을 조회할 수 있는 목록 페이지
- 비로그인 :  자기 소개글들을 읽을 수 있다.
- 로그인 :  자기 소개글들을 읽을 수 있다. 

	(2-1) 페이지 출력
	ㄴ	대표 이미지, 이름, 거주지역, 나이, (희망)포지션, 코멘트 출력	
	ㄴ	작동 : 게시글을 누르면 해당 내용이 출력됨

	(2-2) 글 검색 기능(지역, 포지션)
	ㄴ 작동 : 비로그인/로그인 한 사용자가 검색 칸에 원하는 검색 내용을 적고, 검색 버튼을 누르면 해당 내용이 들어가있는 자기소개게시글들을
			 아래로 출력시킴	


(3) 프로젝트 팀원 찾기 작성 페이지 
- 회원이 자기 소개 글을 작성하는 페이지. 이름, 거주지역, 나이, (희망)포지션, 코멘트를 적는 칸이 있다.
- 비로그인 :  작성 불가. 작성 버튼을 누를 시에 로그인 페이지로 이동한다.
- 로그인 :  게시판 내에 글을 작성할 수 있다.

	(3-1) 글 작성 기능 
	ㄴ	선행 조건 : 로그인이 되어있는 상태
	ㄴ	작동 : <글 쓰기> 버튼을 누르면 작성 페이지로 이동 

	(3-2) 글 작성 제한 조건
	ㄴ	선행 조건 : 글 쓰기 페이지가 로드된 상태
	ㄴ 	대표 이미지 : 필수 입력 x / 1개만 설정 가능 (입력데이터 null 일 경우 관리자 설정 이미지로 대체)
		이름 : 필수 입력 데이터
		거주지역 : 필수 입력 데이터 / 서울특별시, 인천광역시, ....  선택 가능 (서울특별시 누르면 서울 자치구 뜨게하고싶은데 어케함??)
		나이 : 필수 입력 x 
		(희망)포지션 : 필수 입력 데이터 / 프론트엔드/백엔드/기획/마케팅/디자인/기타 (직접입력도 가능하게)
		코멘트 : 필수 입력 데이터 / 500자 제한

	(3-3) 수정 및 삭제 기능
	ㄴ	선행 조건 : 글 쓰기가 완료된 상태
	ㄴ	작동 : <수정> 버튼 클릭시 글 수정 창으로 이동
			  <삭제> 버튼 클릭시 안내 메시지 팝업 후 <예> 버튼 클릭 시에 삭제
