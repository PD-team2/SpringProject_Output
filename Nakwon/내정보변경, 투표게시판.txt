회원정보수정
- 비로그인 : 로그인하지 않은 회원은 사용할 수 없는 기능
- 로그인 : 내정보조회를 통해 회원정보수정 가능

	선행 조건 : 회원가입이 되어있고 아이디와 비밀번호를 입력해 로그인 한 회원만 사용가능해야 함
	
	기본 조건 : 로그인 후 내정보조회 페이지에 들어가 내 정보조회 페이지에서 회원정보 수정 후 저장
	
	제약 조건 : 회원가입이 되어 있지 않은 사람 및 로그인하지 않은 회원은 기능 사용 불가
			아이디는 수정불가
			전체회원조회가 가능(관리자) 할 지라도 전체회원조회를 통해선 회원정보수정 불가
			
	기능 : 내정보조회를 통해 본인 정보 확인 후 아이디를 제외한 비밀번호, 이름, 휴대폰, 이메일 변경 가능

투표 게시판
- 비로그인 : 진행하고 있는 투표 및 진행했던 투표 내용을 확인할 수 있다.
- 로그인 : 진행하고 있는 투표에 직접 투표할 수 있다. 
	
	선행 조건 : 투표 게시판을 만들어 관리자가 투표 게시물을 작성해야 함
			투표의 선택지는 4개로 항상 고정시켜 4개의 컬럼을 가지는 DB테이블 생성 
	
	기본 조건 : 로그인 여부 상관없이 누구나 진행하고 있는 투표 혹은 진행했던 투표 내용을 확인할 수 있다.
			투표결과는 메인화면에 노출시켜 사이트 방문자 누구나 확인 가능
		
	제약 조건 : 로그인 한 사용자만 투표를 할 수 있다.
			다중 투표 및 중복 투표는 불가능하다.
			투표 결과는 투표 진행창이 아닌 메인화면 (혹은 다른 화면)에 노출시킬 투표 결과창을 통해서만 확인 가능하다.
			투표는 관리자만 만들 수 있다.
				
	기능 : 투표는 체크박스 형식으로 체크박스 선택 후 투표완료 창을 누르면 투표 완료
		투표를 할 수 있는 기간을 설정해 기간이 지난 투표는 체크박스를 선택해도 투표 완료 버튼 사용 불가(기간만료된 투표는 투표 완료 버튼을 눌러도 다음 창으로 넘어가지 않음)
		완료된 투표를 차트 형식으로 선택지 비율 메인화면에 노출
		중복불가는 투표를 한 회원은 투표자로 바꾸고 투표를 하지 않은 사람은 회원 혹은 손님으로 바꾸어 투표 중복투표 방지
