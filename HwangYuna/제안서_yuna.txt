(1) 커뮤니티 게시글 신고 기능
- 비로그인 : 신고 불가능
- 로그인 : 신고 가능

	(1-1) 신고 기능
	ㄴ	선행 조건 : 로그인 한 회원, 게시글이 검토 요청 상태가 아니고, 보관함에 있지 않은 글인 경우(상태: F/ 처리: F)
	ㄴ	작동 : 각 게시글마다 표시되어 있는 '신고' 아이콘을 클릭 
				> 신고 사유와 신고하기/취소 선택 가능 (신고 사유 필수)
				> 검토 요청 상태(상태: T/ 처리: T)로 전환되고 게시글이 숨김 상태로 변경 					
				
	ㄴ	관리자 : 관리자 페이지 신고 처리 대기 목록에 신고자/신고사유/게시글 번호가 뜨게됨
			   ex) 신고 신청 회원 정보 	게시글 작성자 회원 정보 		게시글 정보 		상태			처리
					user01				user02			덥다리덥다~~!!		   	T요청   		T숨김
					user05				user06			민트초코존맛탱		    T요청		T숨김
					user05				user06			민트초코는최악이		   	F완료		F재게시
					user05				user06			민트초코는최악이		   	F완료 		T숨김
				
		a. 사유가 허위인 경우 다시 게시판에 보이기(상태: F/ 처리: F)
		b. 신고 사유가 타당할 경우 게시글 숨김 (상태: F/ 처리: T)
			> 신고된 게시글을 작성한 사용자에게 누적 신고 1 회 부여 
			> [삭제 사유와 게시글 제목을 작성자에게 공지]
		
	(1-2) 블랙리스트 기능
	ㄴ 	선행 조건 : 블랙리스트에 넣을 회원은 회원 가입이 되어 있는 상태
	ㄴ	작동 : '신고 현황'에서 게시글 누적 신고가 많이 된 회원들을 리스트로 볼 수 있음. 누적 신고 횟수가 10번이 넘어가는 회원은 
		해당 계정으로 로그인 시 '블랙리스트로 인해 차단 된 계정입니다'메세지 출력
		
(2) 공지사항 
- 비로그인/로그인 : 열람 가능 
	
	2-1) 작성/수정/숨김/삭제
	ㄴ	선행 조건 : 로그인 된 관리자 등급의 사용자 
	ㄴ	숨김 : 게시물 복구 가능, 사용자들은 열람할 수 없고 숨김 해제시 다시 목록에 나타남
	ㄴ	삭제 : 게시글 삭제시 복구 불가능
